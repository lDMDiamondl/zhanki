# Zhanki

Zhanki는 Spaced Repetition System (SRS) 알고리즘을 활용한 웹 기반 중국어 단어 암기 도구입니다. Anki의 학습 로직을 벤치마킹하여 효율적인 암기 프로세스를 제공합니다.

## 주요 기능

1. Anki 스타일 SRS 알고리즘
- 새로운 카드 학습, 이미 암기한 카드의 복습, 틀린 카드의 재학습 단계를 구분하여 관리합니다.
- 사용자의 피드백(다시, 어려움, 괜찮음, 쉬움)에 따라 복습 간격을 유동적으로 조정합니다.

2. 학습 관리
- 하루 새 단어 학습량 제한 설정을 통해 무리한 학습을 방지합니다.
- 오늘의 학습 카드가 부족할 경우 자동으로 새 단어를 큐에 추가합니다.

3. 데이터 로컬 저장 및 관리
- JSON 형식을 통해 학습 데이터를 백업하거나 다른 기기로부터 가져올 수 있습니다.

4. 단어장 관리
- 카드 목록 탭에서 암기 중인 단어들을 확인하고 직접 수정하거나 삭제할 수 있습니다.

## 알고리즘 상세

본 프로젝트는 SM-2 알고리즘을 기반으로 개조된 로직을 사용합니다.

- 다시: 학습 단계로 돌아가 1분 후 재노출됩니다.
- 어려움: 현재 간격을 유지하거나 소폭 늘리며, 난이도 계수가 감소합니다.
- 괜찮음: 난이도 계수에 따라 복습 간격이 지수적으로 증가합니다.
- 쉬움: 단계를 즉시 건너뛰고 장기 복습으로 넘어가며 난이도 계수가 증가합니다.
